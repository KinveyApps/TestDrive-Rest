TestDrive-Rest
==============

Examples of using Kinvey's service in various languages

* Objective-C
    * Interact directly with REST API Using NSURLSession 
    * Requires XCode 5+, iOS 7+
* Swift
    * Requires XCode 6+, iOS 8+
